# KingDomino

**To Play on Windows**

*Download the .jar file 
*On the folder containing the file, open powershell (can be dont by pressing shift and right mouse click)
*Type java -jar kingDomino.jar
*Press Enter
*Enjoy!

**To Play on MACOS**

-Download the .jar file 
-Navigate to the folder containing the .jar file
-Type java -jar kingDomino.jar
-Press Enter
-Enjoy!
